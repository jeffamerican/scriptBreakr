It takes raw screenplay text as input and extracts scene information, including scene headings, characters, location, and more. Additionally, it utilizes OpenAI's GPT-3 model to extract information about characters, stunts, and extras from the scene actions.

  • Parse raw screenplay text and extract scene information
  • Display scene information in a Treeview
  • Edit scene information in a user-friendly interface
  • Analyze actions in the scenes using OpenAI's GPT-3 model
  • Extract character names, stunt descriptions, and extras information from the scene actions
